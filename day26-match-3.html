💎 Match-3 — Swap Gems, Clear Lines of Three, Cascade the Falls

Swap two touching gems to line up three or more of a colour.

Tap a gem to pick it up, then tap a gem directly next to it to swap the pair. The swap is only allowed if it forms a run of three or more of the same colour in a row or a column — if it does not, the gems slide back and it costs you nothing. Matched runs pop and vanish; the gems above them fall down to fill the gaps, and fresh gems drop in from the top. If the new arrangement lines up another match, it clears automatically — a cascade — and each link in the chain scores more than the last. When the board has no swap left that makes a match, it reshuffles itself so you are never stuck.

The whole board is an 8×8 grid of small integers, one per gem colour. Every rule you see is one short function over that grid: a scan that walks each row and each column looking for runs of three or more equal values, a column-collapse that drops the survivors and tops up with new gems, and a loop that repeats clear → fall → refill until nothing matches. A swap is legal only when running the match scan on the swapped board finds something. All of it is vanilla JavaScript running offline — no images, no server, no libraries.

🔨 Build it yourself

    
An 8×8 grid of gem-colour integers, a generator that seeds it with no accidental matches, one scan that flags every run of three or more, a renderer that draws the grid and the selection, a click handler that swaps adjacent gems only when the swap makes a match, a column-collapse that drops survivors and refills from the top, and a cascade loop that chains combos and scores them. About 120 lines of vanilla JS — copy each block in order.

    
    

      
  1. 
        
    1
    The grid, the gems, and a clean start
    
        
    The board is an N×N array where each cell holds a small integer naming a gem colour. Fill it randomly, but never place a gem that would already complete a run of three with the two cells to its left or the two cells above — that guarantees the opening board has no free matches waiting to clear.
    
        
    const N = 8, T = 6;              // 8×8 board, 6 gem colours
let grid = [];                  // grid[r][c] = 0..T-1
const rnd = () => Math.floor(Math.random() * T);

function fillNoMatch(){
  grid = Array.from({length:N}, () => Array(N).fill(-1));
  for (let r=0;r<N;r++) for (let c=0;c<N;c++){
    let t;
    do { t = rnd(); }
    while ((c>=2 && grid[r][c-1]===t && grid[r][c-2]===t) ||   // no 3-in-row
           (r>=2 && grid[r-1][c]===t && grid[r-2][c]===t));    // no 3-in-col
    grid[r][c] = t;
  }
}

  2. 
        
    2
    Find every match in one scan
    
        
    A match is a run of three or more equal colours in a row or a column. Walk each row counting the current run length, and whenever the run breaks flag the cells you just passed if it reached three. Do the same down each column. A boolean mark grid collects every matched cell, so overlapping row and column runs are handled for free.
    
        
    function findMatches(){
  const mark = Array.from({length:N}, () => Array(N).fill(false));
  const flag = (get, set) => {           // scan one line direction
    let run = 1;
    for (let i=1;i<=N;i++){
      if (i<N && get(i)===get(i-1) && get(i)>=0) run++;
      else { if (run>=3) for (let k=i-run;k<i;k++) set(k); run = 1; }
    }
  };
  for (let r=0;r<N;r++) flag(i=>grid[r][i], k=>mark[r][k]=true);   // rows
  for (let c=0;c<N;c++) flag(i=>grid[i][c], k=>mark[k][c]=true);   // cols
  const out = [];
  for (let r=0;r<N;r++) for (let c=0;c<N;c++) if (mark[r][c]) out.push({r,c});
  return out;                            // [] means "no matches"
}

    3
    Draw the grid and the selection
    
        
    Render is a pure function of the grid: lay N columns of fixed-size tiles, drop the gem glyph into each, and give the currently picked-up cell a highlight class. Every cell carries its own click handler so a tap knows exactly which gem it is.
    
        
    const GEMS = ["🔴","🟠","🟡","🟢","🔵","🟣"];
function render(){
  board.style.gridTemplateColumns = `repeat(${N}, var(--cs))`;
  board.innerHTML = "";
  for (let r=0;r<N;r++) for (let c=0;c<N;c++){
    const d = document.createElement("div");
    const picked = sel && sel.r===r && sel.c===c;
    d.className = "gem" + (picked ? " sel" : "");
    d.textContent = GEMS[grid[r][c]];
    d.onclick = () => clickCell(r,c);
    board.appendChild(d);
  }
}
    
      
    4. 

      
  4. 
        
    4
    Tap, swap, and undo illegal swaps
    
        
    The first tap picks a gem up; a second tap on an adjacent gem swaps the two. Swap the values, then ask findMatches: if it finds anything, keep the swap and start resolving; if not, swap the pair straight back — an illegal move that costs nothing. Only legal swaps count as a move.
    
        
    let sel = null, busy = false;
const swapCells = (a,b,c,d) => { const t=grid[a][b]; grid[a][b]=grid[c][d]; grid[c][d]=t; };
const adjacent  = (a,b) => Math.abs(a.r-b.r) + Math.abs(a.c-b.c) === 1;

function clickCell(r,c){
  if (busy) return;
  if (!sel){ sel = {r,c}; render(); return; }
  if (sel.r===r && sel.c===c){ sel = null; render(); return; }   // tap again = drop
  if (adjacent(sel, {r,c})){
    const a = sel; swapCells(a.r,a.c,r,c);
    if (findMatches().length){                 // legal — it makes a match
      moves++; sel = null; busy = true; cascade = 0; render();
      setTimeout(resolve, 180);
    } else {                                    // illegal — slide back
      swapCells(a.r,a.c,r,c); sel = null; render();
    }
  } else { sel = {r,c}; render(); }             // pick a different gem
}
    
      
    5. 

      
  5. 
        
    5
    Gravity: collapse and refill each column
    
        
    After clearing, work one column at a time. Gather the gems that survived from the bottom up, stack them back at the bottom, then fill whatever is left at the top with brand-new random gems. That single pass is both the fall and the refill.
    
        
    function applyGravity(){
  for (let c=0;c<N;c++){
    const keep = [];
    for (let r=N-1;r>=0;r--) if (grid[r][c] !== -1) keep.push(grid[r][c]);
    let r = N-1;
    for (const v of keep) grid[r--][c] = v;     // survivors fall to the bottom
    for (; r>=0; r--) grid[r][c] = rnd();        // new gems drop in on top
  }
}
    
      
    6. 

      
  6. 
        
    6
    The cascade loop and scoring
    
        
    Resolving is a loop: find matches, clear them, apply gravity, and go again — because the fresh gems may line up new runs. Each pass increases the cascade level, and points scale with it, so a four-deep chain from one swap is worth far more than four separate swaps. When a pass finds nothing, the board is stable.
    
        
    let score = 0, moves = 0, cascade = 0;
function resolve(){
  const m = findMatches();
  if (!m.length){                       // stable — hand control back
    busy = false;
    if (!hasMove()) shuffle();           // (step 7)
    return;
  }
  cascade++;
  score += m.length * 10 * cascade;      // deeper chains pay more
  for (const p of m) grid[p.r][p.c] = -1;   // clear
  render();
  setTimeout(() => { applyGravity(); render(); setTimeout(resolve, 240); }, 240);
}
    
      
    7. 

      
  7. 
        
    7
    Never get stuck: detect dead boards and shuffle
    
        
    A board is dead when no single legal swap makes a match. Test it by trying every right-swap and down-swap, checking, then undoing. If none work, keep re-shuffling the existing gems until the board both has a move and has no free matches — so the player always has something to do.
    
        
    function hasMove(){
  for (let r=0;r<N;r++) for (let c=0;c<N;c++){
    for (const [dr,dc] of [[0,1],[1,0]]){
      const nr=r+dr, nc=c+dc; if (nr>=N || nc>=N) continue;
      swapCells(r,c,nr,nc);
      const ok = findMatches().length > 0;
      swapCells(r,c,nr,nc);              // always undo the probe
      if (ok) return true;
    }
  }
  return false;
}
function shuffle(){
  do {
    const flat = grid.flat();
    for (let i=flat.length-1;i>0;i--){ const j=Math.random()*(i+1)|0; [flat[i],flat[j]]=[flat[j],flat[i]]; }
    let k=0; for (let r=0;r<N;r++) for (let c=0;c<N;c++) grid[r][c]=flat[k++];
  } while (findMatches().length || !hasMove());   // legal AND playable
  render();
}
