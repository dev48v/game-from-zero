🔴 Connect Four — Beat the Minimax AI

Drop a red disc. The AI drops yellow. Get four in a row.

You play red, the computer plays yellow and thinks ahead with minimax. Line up four — across, down, or on either diagonal — before it does. It will block your threats and pounce on any win it can find.

Turn: You (red)

🔨 Build it yourself

    
A 2D grid, a drop rule, a four-direction win check, and a minimax search with alpha-beta pruning. ~120 lines of vanilla JS — copy each block in order.

    
    

      
  1. 
        
    1
    The board as a 2D grid
    
        
    Connect Four is 7 columns by 6 rows. We store it as an array of 6 rows, each an array of 7 cells. 0 is empty, 1 is you (red), 2 is the AI (yellow). Row 0 is the bottom so a piece "falls" to the lowest empty cell.
    
        
    
          
    const COLS = 7, ROWS = 6;
const HUMAN = 1, AI = 2;
let board, turn, over;

function makeBoard(){
  return Array.from({length: ROWS}, () => Array(COLS).fill(0));
}
    
          
        
    
      
    2. 

      
  2. 
        
    2
    Dropping a disc
    
        
    A move is just a column. Walk up from the bottom row and place the disc in the first empty cell. If the top cell is full the column is illegal. validMoves lists every column that still has room.
    
        
    
          
    function drop(b, col, player){
  for (let r = 0; r < ROWS; r++){
    if (b[r][col] === 0){ b[r][col] = player; return r; }  // lowest empty
  }
  return -1;                       // column is full
}
function validMoves(b){
  const m = [];
  for (let c = 0; c < COLS; c++) if (b[ROWS-1][c] === 0) m.push(c);
  return m;
}
    
          
        
    
      
    3. 

      
  3. 
        
    3
    Win check in four directions
    
        
    From every cell, look in four directions — right, up, and both diagonals — and count four matching discs in a line. We return the four cells so the UI can highlight the winning row. The four directions cover horizontal, vertical, and both diagonals once each.
    
        
    
          
    const DIRS = [[0,1],[1,0],[1,1],[1,-1]];   // →  ↑  ↗  ↘
function winner(b){
  for (let r = 0; r < ROWS; r++)
    for (let c = 0; c < COLS; c++){
      const p = b[r][c]; if (!p) continue;
      for (const [dr,dc] of DIRS){
        const line = [[r,c]];
        for (let k = 1; k < 4; k++){
          const nr = r+dr*k, nc = c+dc*k;
          if (nr<0||nr>=ROWS||nc<0||nc>=COLS||b[nr][nc]!==p) break;
          line.push([nr,nc]);
        }
        if (line.length === 4) return { player:p, line };
      }
    }
  return null;
}
    
          
        
    
      
    4. 

      
  4. 
        
    4
    Scoring a window of four
    
        
    When the search can't see a win yet, it judges a position by sliding a 4-cell "window" over the board. Three of mine with a gap is great; three of yours with a gap is dangerous, so we subtract for it. Summing every window gives one number: how good the board is for the AI.
    
        
    
          
    function scoreWindow(w, me){
  const foe = me === AI ? HUMAN : AI;
  const my = w.filter(x => x === me).length;
  const op = w.filter(x => x === foe).length;
  const empty = w.filter(x => x === 0).length;
  if (my === 4) return 100000;
  if (my === 3 && empty === 1) return 50;
  if (my === 2 && empty === 2) return 5;
  if (op === 3 && empty === 1) return -80;   // block their threat
  return 0;
}
    
          
        
    
      
    5. 

      
  5. 
        
    5
    The evaluation function
    
        
    Run scoreWindow over every horizontal, vertical, and diagonal window on the board and add them up. A small bonus for the centre column helps, because central pieces sit on more winning lines. One number summarises the whole position.
    
        
    
          
    function evaluate(b){
  let s = 0;
  for (let r = 0; r < ROWS; r++)
    if (b[r][3] === AI) s += 3;                 // prize the centre
  // every horizontal, vertical and diagonal window of 4
  for (const w of allWindows(b)) s += scoreWindow(w, AI);
  return s;
}

    6
    Minimax with alpha-beta pruning
    
        
    The AI imagines the future: on its turn it picks the move with the highest score (maximise), and assumes you reply with the lowest (minimise). It recurses until it finds a win, a draw, or runs out of depth, then trusts evaluate. Alpha-beta tracks the best each side is guaranteed and stops exploring branches that can't beat it.
    
        
    
          
    function minimax(b, depth, alpha, beta, maximizing){
  const win = winner(b);
  if (win) return { score: win.player === AI ? 1e6+depth : -1e6-depth };
  if (depth === 0 || validMoves(b).length === 0)
    return { score: evaluate(b) };

  const moves = validMoves(b);
  let best = maximizing ? -Infinity : Infinity, bestCol = moves[0];

  for (const col of moves){
    const nb = b.map(row => row.slice());        // copy the board
    drop(nb, col, maximizing ? AI : HUMAN);
    const val = minimax(nb, depth-1, alpha, beta, !maximizing).score;

    if (maximizing){
      if (val > best){ best = val; bestCol = col; }
      alpha = Math.max(alpha, best);
    } else {
      if (val < best){ best = val; bestCol = col; }
      beta = Math.min(beta, best);
    }
    if (alpha >= beta) break;                     // prune
  }
  return { score: best, col: bestCol };
}

  7. 
        
    7
    The AI takes its turn
    
        
    Ask minimax for the best column at depth 4, drop a yellow disc there, then check for a win or a full-board draw. We run it after a short delay so the player sees their own move land first.
    
        
    
          
    function aiMove(){
  const { col } = minimax(board, 4, -Infinity, Infinity, true);
  drop(board, col, AI);
  render();
  const w = winner(board);
  if (w) return finish(w);
  if (validMoves(board).length === 0) return finish(null);  // draw
  turn = HUMAN;
}
    
          
        
    
      
    8. 

      
  8. 
        
    8
    Wire up the UI
    
        
    Render the grid as seven clickable columns. A click drops the human disc; if nobody has won and it's still the AI's turn, hand control to aiMove after a beat. Lock input while the AI thinks or the game is over. Double-click the file — no server needed.
    
        
    
          
    function play(col){
  if (over || turn !== HUMAN) return;
  if (drop(board, col, HUMAN) === -1) return;   // column full
  render();
  const w = winner(board);
  if (w) return finish(w);
  if (validMoves(board).length === 0) return finish(null);
  turn = AI;
  setTimeout(aiMove, 250);                       // let the AI reply
}
function reset(){ board = makeBoard(); turn = HUMAN; over = false; render(); }
reset();
