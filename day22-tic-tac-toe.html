⭕ Tic-Tac-Toe — the AI That Never Loses

Try to beat it. On "Unbeatable" you can't — the best you'll ever get is a draw.

This computer opponent plays with minimax: before every move it mentally plays out every possible continuation of the game to the end, scores each ending (win, lose, draw), and assumes both sides play perfectly. Because tic-tac-toe is small enough to search completely, the AI always finds a move that can never lead to a loss. Switch to Easy to see it play random moves instead, or flip whether you're X or O (X always moves first).

🔨 Build it yourself

    
A board held as a 9-slot array, a winner check over the eight winning lines, a recursive minimax() that scores every possible ending, a move-picker that chooses the highest-scoring move, and a bit of UI glue. ~90 lines of vanilla JS — copy each block in order.

    
    

      
  1. 
        
    1
    Model the board
    
        
    The board is one array of nine cells, indexed 0–8 (top-left to bottom-right). Each cell holds "X", "O", or "" for empty. This flat model makes it trivial to copy the board for the AI's what-if search and to list the empty cells that are the legal moves.
    
        
    let board = ["","","","","","","","",""];  // 9 cells, 0..8
const HUMAN = "X", AI = "O";                // who is who
const empties = b => b.map((v,i)=>v?-1:i).filter(i=>i>=0);

  2. 
        
    2
    Detect a winner
    
        
    There are exactly eight ways to win: three rows, three columns, two diagonals. We list those triples once, then check whether any line holds the same non-empty mark in all three cells. This function is the AI's "terminal test" — it tells the search when a game has ended and who won.
    
        
    const LINES = [[0,1,2],[3,4,5],[6,7,8],   // rows
               [0,3,6],[1,4,7],[2,5,8],   // cols
               [0,4,8],[2,4,6]];          // diagonals
function winner(b){
  for (const [a,c,d] of LINES)
    if (b[a] && b[a] === b[c] && b[a] === b[d]) return b[a];
  return b.every(Boolean) ? "draw" : null;   // full board = draw
}

  3. 
        
    3
    Minimax — score every ending
    
        
    Minimax recursively plays out the game. On the AI's turn it wants the maximum score; on your turn it assumes you'll pick the minimum (best for you). At a finished game it returns +10 if the AI won, −10 if you won, 0 for a draw. We subtract the search depth from a win and add it to a loss so the AI prefers to win quickly and lose slowly.
    
        
    function minimax(b, isAI, depth){
  const w = winner(b);
  if (w === AI)    return  10 - depth;   // win sooner = better
  if (w === HUMAN) return -10 + depth;   // lose later = less bad
  if (w === "draw") return 0;

  const scores = empties(b).map(i => {
    b[i] = isAI ? AI : HUMAN;            // try the move
    const s = minimax(b, !isAI, depth+1);// opponent responds
    b[i] = "";                          // undo
    return s;
  });
  return isAI ? Math.max(...scores) : Math.min(...scores);
}

    4
    Pick the best move
    
        
    To choose a move, the AI tries each empty cell, runs minimax on the resulting position (now it's your turn, so isAI = false), and keeps the move with the highest score. Because minimax already looked all the way to the end of every line, this greedy pick of the top-scoring move is optimal play.
    
        
    function bestMove(b){
  let best = -Infinity, move = -1;
  for (const i of empties(b)){
    b[i] = AI;
    const s = minimax(b, false, 0);   // human replies next
    b[i] = "";
    if (s > best){ best = s; move = i; }
  }
  return move;
}

    Wire up turns & rendering
    
        
    On a human click we place the mark, check for a result, and if the game is still going, let the AI move (minimax on Unbeatable, a random empty cell on Easy). Rendering just reads the array into nine cells and highlights the winning line. Alpha-beta pruning would speed a bigger game up, but tic-tac-toe's ~9! ≈ 362,880 leaf nodes search instantly as-is.
    
        
    function humanMove(i){
  if (board[i] || winner(board)) return;
  board[i] = HUMAN; render();
  if (winner(board)) return finish();
  const ai = difficulty === "hard"
    ? bestMove(board)
    : empties(board)[Math.random()*empties(board).length|0];
  board[ai] = AI; render(); finish();
}
